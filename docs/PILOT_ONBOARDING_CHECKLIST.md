# NexusAI V1 Pilot Onboarding Checklist

## 1) Sponsor and Scope
- Confirm executive sponsor (CEO/COO/CBO equivalent).
- Confirm 6-8 week pilot window.
- Lock outcomes: decision clarity, risk visibility, recommendation quality.

## 2) Workspace Setup
- Create tenant + workspace.
- Assign role mapping (CEO, COO, CBO/Strategy).
- Configure user allowlist.

## 3) Source Bundle (Docs + Comms)
- Connect Google Drive or SharePoint pilot folders.
- Connect Slack (mention-based) channels/threads.
- Validate upload ingestion path for PDF/DOCX/PPTX/XLSX.

## 4) Governance Policy
- Set quarantine threshold (default 0.55).
- Require approval for recommendation promotion.
- Confirm restricted-data masking for Slack summaries.

## 5) Validation
- Run ingestion status check.
- Generate first CEO/COO/CBO dashboards.
- Produce first Weekly Executive Brief and Decision Memo.
