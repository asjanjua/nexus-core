# LM Studio Benchmark Kit

This folder contains the local LLM feasibility and optimization workflow for Apple Silicon systems.

- Runbook: `LM_STUDIO_MOE_OPTIMIZATION.md`
- Results: `results/`

## Quick start

From repo root:

```bash
npm run lmstudio:all
```

Or run stages individually:

```bash
npm run lmstudio:capability
npm run lmstudio:benchmark
npm run lmstudio:moe-probe
```
