# NexusAI V1 Pilot Success Scorecard

## KPI Targets
- Executive briefs generated: >= 3
- Recommendations created: >= 20
- Recommendation evidence coverage: >= 80%
- Decisions captured with owner/rationale/source refs: >= 10
- Unprovenanced dashboard cards: 0
- Restricted-data leak incidents in Slack summaries: 0

## Operating Checks
- Dashboard freshness and confidence metadata visible.
- Quarantine queue reviewed weekly.
- Approval queue cycle time tracked.

## Sponsor Sign-Off Questions
- Did Nexus reduce time to understand status and risk?
- Did decision quality improve with evidence links?
- Would sponsor continue into expanded phase?
