# Nexus Launch Kit

This folder collects the core public-launch materials for Nexus Core.

## Included
- `landing-page.md` - a short web/blog landing-page draft
- `launch-post-template.md` - a social post template for X, LinkedIn, or a newsletter
- `assets/` - reusable visual assets for screenshots, cards, and demo frames

## Recommended Launch Order
1. Publish the release notes and README updates.
2. Post the landing-page copy as a blog or dev log.
3. Share the launch post on X and LinkedIn.
4. Add the visual assets into the post or blog header.

## Best Practice
Keep the launch message simple:
- what Nexus is
- who it helps
- how to try it
- why it is different
